cuRE: CUDA R extensions

Harshavardhan Pandit and Ravi Chandra Bandlamudi

Summary

We will be developing a library of R interfaces for optimized CUDA functions. The aim is to enable faster computation of commonly used R functions.

Background

R is a popular open-source platform used statistical computing. However, peforming statistical and/or linear algebraic operations in R is very time consuming. Implementing particular commonly-used functions in CUDA and accessing these functions from R, can significantly improve the performance of R functions (eg: https://developer.nvidia.com/hiplar and h http://heather.cs.ucdavis.edu/~matloff/rth.html#down). Linear algebraic functions such as matrix multipilcation, least squares etc stand to benefit from such GPU acceleration.
